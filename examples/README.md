# Examples

This directory contains various examples of how h3mtxt can be used as a library.

Some examples are trivial and only serve to demonstrate the API: for example, [disable_fly_in_mage_guilds](disable_fly_in_mage_guilds) is not very useful, considering that there can be at most 48 towns on the Adventure Map.

[Minimap](Minimap) and [draw_minimap](draw_minimap), on the other hand, are fully functional.
