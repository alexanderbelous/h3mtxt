# A library for creating minimaps for HoMM3 maps.

A minimap is the preview image of the HoMM3 map, which is displayed in the game and in the Map Editor. A lot of websites that publish HoMM3 maps also display such images.

This library demonstrates how minimaps can be created using h3mtxt. This library is used by [examples/draw_minimap](../draw_minimap), which is a program for creating minimaps.

The library uses a thirdparty library [lodepng](https://github.com/lvandeve/lodepng) for saving PNG images.
